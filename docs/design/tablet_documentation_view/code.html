Understanding Workspaces

Workspaces are the fundamental unit of organization in Choorai. They allow you to group your projects, assets, and team members effectively into isolated environments.

Architecture
lightbulb

Pro Tip: Unlimited Workspaces

You can create unlimited workspaces on the Pro plan to better organize different departments or client projects without incurring extra costs per seat.

Creating a Workspace

Creating a new workspace is straightforward. Administrators can create workspaces from the main dashboard. When you create a workspace, you automatically become its owner.

  1. 1

    Navigate to Dashboard

    Click on the logo in the top left corner to return to your main view.

  2. 2

    Select "New Workspace"

    Find the plus icon (+) next to your existing workspaces list in the sidebar.

  3. 3

    Configure Details

    Enter a unique name and select an icon/color for easy identification.

workspace_config.json 
{
  "name": "Marketing Team",
  "type": "department",
  "visibility": "private",
  "members_allowed": true
}

Managing Access

Once created, you can invite members via email. It is recommended to use the "Invite Link" feature for large teams to simplify the onboarding process.

warning
Important Note

Workspaces cannot be merged once created. Please plan your structure carefully before creating multiple environments.
