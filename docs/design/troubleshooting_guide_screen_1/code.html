Troubleshooting & Solutions

Find solutions for common build and deployment errors instantly.

Trending:
Critical CODE: 403

Access to XMLHttpRequest blocked by CORS policy

info The Cause

Your frontend is trying to request data from a backend hosted on a different domain, and the browser blocked it for security reasons.

check_circle The Solution
// In your backend configuration
Access-Control-Allow-Origin: "https://your-frontend.com"

Or navigate to Settings > Security > Allowed Origins to add your domain.

Warning Runtime

process.env.API_KEY is undefined

info The Cause

The application cannot read the `API_KEY` environment variable. This usually happens when the variable is not set in the cloud dashboard.

check_circle The Solution
  1. Go to your Project Dashboard.
  2. Navigate to Settings > Environment Variables.
  3. Add Key: API_KEY and your value.
  4. Redeploy the application.
Critical Build

Error: Command "npm run build" exited with 1

info The Cause

The build script failed. This is often caused by syntax errors in your code, missing dependencies, or type errors in TypeScript.

check_circle The Solution
Step 1: Check the logs immediately above this error line.
Step 2: Look for "Module not found" or "SyntaxError".
Step 3: Run npm install locally to ensure dependencies are correct.
