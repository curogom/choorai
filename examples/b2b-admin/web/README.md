# B2B Admin Web (React + Vite)

> React + Vite + TanStack Query 기반 프론트엔드

## 설정 예정

- [ ] Vite + React 프로젝트 초기화
- [ ] TanStack Query 설정
- [ ] Health 표시 페이지
- [ ] Projects 목록/생성 UI
- [ ] 환경 변수 설정

## 개발

```bash
# 의존성 설치
npm install

# 개발 서버
npm run dev

# 빌드
npm run build
```

## 배포

Cloudflare Pages로 배포됩니다.
